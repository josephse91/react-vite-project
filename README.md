## Run Locally

Clone the project

```bash
  git clone https://github.com/josephse91/react-vite-project.git
```

Go to the project directory

```bash
  cd teamwork-dreamwork
```

Install dependencies

```bash
  npm install
```

```bash
  npm run start (This will start the server)
```
